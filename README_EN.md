# 🎯 HuChengfeng Companion — VS Code Extension

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A VS Code companion extension with a **direct-focus + practical execution** tone.

## Features

- Chat panel: `HuChengfeng Focus Room`
- Pomodoro focus & break cycle
- Diagnostic-based code health summary
- Save-triggered tips and idle reminders
- Daily counters (pomodoros / messages / saves)
- **Persona strategy controls**: `personaMode` (`neutral` / `standard` / `intense`) and `safeMode`

## Commands

- `HuChengfeng: Open Focus Stage`
- `HuChengfeng: Start Pomodoro`
- `HuChengfeng: Stop Pomodoro`
- `HuChengfeng: Give Me Progress Advice`
- `HuChengfeng: Review My Code`
- `HuChengfeng: End Today`
- `HuChengfeng: Sharp Line`
- `HuChengfeng: City Rule (Work Edition)`
- `HuChengfeng: Give Me a Label`

## Run from source

```bash
npm install
npm run compile
```

Press **F5** to launch the Extension Development Host.
